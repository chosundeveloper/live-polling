🎮 사용 방법

  1. 디스플레이 화면 열기 - 위 버튼을 클릭하여 프로젝터/대형 화면에 표시
  2. 질문 전환 - 키보드 화살표(←, →) 또는 화면 하단 버튼으로 질문 변경
  3. QR 코드 스캔 - 참가자들이 스마트폰으로 QR 코드 스캔
  4. 답변 제출 - 100초 안에 답변 작성 후 제출
  5. 실시간 확인 - 화면에 답변이 예쁘게 애니메이션으로 나타남
